# Smart Product Export

A powerful WordPress plugin for exporting WooCommerce product SKUs based on various filtering criteria.

## Description

Smart Product Export allows you to easily export product SKUs from your WooCommerce store in a comma-separated format. Filter products by SKU, ID, category, tag, attributes, or export all products at once. Perfect for inventory management, reporting, and bulk operations.

## Features

- **Multiple Filter Options:**
  - Export all products
  - Filter by SKU (partial match)
  - Filter by Product ID(s)
  - Filter by Category
  - Filter by Tag
  - Filter by Product Attributes

- **Variation Support:** Option to include product variations in the export
- **User-Friendly Interface:** Clean, intuitive admin interface
- **One-Click Copy:** Copy results to clipboard with a single click
- **Real-time Results:** AJAX-powered instant results without page reload
- **Comma-Separated Output:** Ready-to-use SKU format: `SKU1, SKU2, SKU3`

## Requirements

- WordPress 5.8 or higher
- WooCommerce 5.0 or higher
- PHP 7.4 or higher
